# 🧪 QA - API de Usuários

Este repositório contém os testes e a documentação da API de gerenciamento de usuários desenvolvida em grupo.

## 🔗 API utilizada

Projeto original desenvolvido por Luiz:
https://github.com/luizcodee/User-API

## 👩‍💻 Meu papel no projeto

* Criação de casos de teste
* Execução de testes manuais
* Validação de regras de negócio
* Documentação de cenários
* Registro de bugs

## 🧪 Tipos de testes realizados

* Testes funcionais
* Testes de validação de dados
* Testes de API (Postman)

## 🛠 Ferramentas utilizadas

* Postman
* GitHub
* Python (API base)

## 📁 Estrutura do projeto

* `/docs` → documentação e casos de teste
* `/bugs` → registro de falhas encontradas
* `/postman` → collections de testes
* `/docs/evidencias` → prints dos testes

## 🎯 Objetivo

Garantir a qualidade da API, validando funcionalidades como cadastro, consulta, atualização e exclusão de usuários.
